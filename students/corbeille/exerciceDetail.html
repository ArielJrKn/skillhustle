Problem Description

Given a non-empty binary tree, find the maximum path sum.

For this problem, a path is defined as any sequence of nodes from some starting node to any node in the tree along the parent-child connections. The path must contain at least one node and does not need to go through the root.

Example:

    Input: [1,2,3]
        1
       / \
      2   3

    Output: 6
    Explanation: The optimal path is 2 -> 1 -> 3 with a sum of 2 + 1 + 3 = 6.

Example 2:

    Input: [-10,9,20,null,null,15,7]
       -10
       / \
      9  20
        /  \
       15   7

    Output: 42
    Explanation: The optimal path is 15 -> 20 -> 7 with a sum of 15 + 20 + 7 = 42.

Constraints:

  • The number of nodes in the tree is in the range [1, 3 * 104].
  • -1000 ≤ Node.val ≤ 1000
  • Time complexity should be O(n) where n is the number of nodes.

Hint:

Think about using a recursive approach. For each node, calculate the maximum path sum that goes through that node, considering both its left and right subtrees.

Solution:

    /**
     * Definition for a binary tree node.
     * function TreeNode(val, left, right) {
     *     this.val = (val===undefined ? 0 : val)
     *     this.left = (left===undefined ? null : left)
     *     this.right = (right===undefined ? null : right)
     * }
     */
    /**
     * @param {TreeNode} root
     * @return {number}
     */
    var maxPathSum = function(root) {
        let maxSum = -Infinity;
        
        // Helper function to find max path sum with the node as highest point
        const findMaxPathSum = (node) => {
            if (!node) return 0;
            
            // Get max path sum from left and right subtrees
            // If path sum is negative, we don't include it
            const leftMax = Math.max(0, findMaxPathSum(node.left));
            const rightMax = Math.max(0, findMaxPathSum(node.right));
            
            // Current path sum is node value plus max path sum from left and right
            const currentPathSum = node.val + leftMax + rightMax;
            
            // Update global max sum if current path sum is greater
            maxSum = Math.max(maxSum, currentPathSum);
            
            // Return max path sum for parent call
            // We can only choose one path (left or right) when returning to parent
            return node.val + Math.max(leftMax, rightMax);
        };
        
        findMaxPathSum(root);
        return maxSum;
    };
